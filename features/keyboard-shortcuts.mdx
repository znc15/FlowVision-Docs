# 快捷键

## 编辑操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `Ctrl+C` | 复制选中元素 |
| `Ctrl+V` | 粘贴 |
| `Delete` | 删除选中节点/连线 |
| `Ctrl+A` | 全选 |

## 视图操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+E` | 导出 PNG |
| `滚轮` | 缩放画布 |
| `拖拽空白` | 平移画布 |

## 聊天操作

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 发送消息 |
| `Shift+Enter` | 消息换行 |
| `/` | 打开命令面板 |
