# AI 智能生成

FlowVision 的核心功能之一 — 用自然语言描述你的需求，AI 自动生成完整的流程图。

## 使用方式

1. 在左侧聊天面板输入需求描述
2. AI 分析需求并生成流程图结构
3. 生成结果实时显示在右侧画布上
4. 可继续对话优化或修改生成结果

## 示例提示词

```
请帮我画一个用户注册登录的流程图，包含：
- 用户输入邮箱和密码
- 验证邮箱格式
- 检查密码强度
- 发送验证邮件
- 激活账号
```

## 支持的 AI 模型

| 提供商 | 推荐模型 | 特点 |
|--------|----------|------|
| OpenAI | `gpt-4o` | 综合能力强，理解复杂流程 |
| Anthropic | `claude-sonnet-4-20250514` | 逻辑推理出色，结构清晰 |
| DeepSeek | `deepseek-chat` | 性价比高，中文理解好 |
| Ollama | 本地模型 | 离线可用，数据隐私 |

## 流式生成

FlowVision 支持 SSE 流式传输，AI 生成过程实时渲染到画布：

- 节点逐步出现，可观察生成过程
- 生成过程中可随时中断
- 生成完成后自动执行「自动布局」

## 思考模式

部分模型支持思考模式（如 DeepSeek-R1），可在设置中开启：

- 开启后 AI 会先分析需求、规划结构
- 思考过程会显示在聊天面板
- 最终产出更精确的流程图结构
