# 导出与导入

## 导出格式

FlowVision 支持将流程图导出为多种格式：

| 格式 | 说明 |
|------|------|
| **PNG** | 高清图片，适合分享和文档 |
| **JSON** | 完整数据结构，可再次导入 |
| **Markdown** | 文本描述，适合文档嵌入 |
| **Mermaid** | Mermaid 格式文本，可在支持 Mermaid 的平台渲染 |

## 导出方式

- **快捷键**：`Ctrl+E` 导出 PNG
- **菜单**：工具栏 → 导出按钮
- **斜杠命令**：`/export`
- **MCP**：通过 `export_graph` 工具导出

## 导入

支持导入 FlowVision JSON 格式的流程图文件：

1. 点击工具栏导入按钮
2. 选择 `.json` 文件
3. 流程图加载到画布

也可通过 MCP 的 `import_graph` 工具从外部导入。
