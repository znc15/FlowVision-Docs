# 可视化编辑

FlowVision 提供专业的流程图画布编辑器，基于 React Flow 构建。

## 节点类型

支持 12 种节点类型，覆盖常见流程图场景：

| 类型 | 用途 |
|------|------|
| 默认节点 | 通用处理步骤 |
| 输入节点 | 流程起点 |
| 输出节点 | 流程终点 |
| 条件节点 | 判断分支 |
| 数据节点 | 数据存储/读取 |
| 注释节点 | 补充说明 |
| 分组容器 | 归类多个节点 |
| 自定义节点 | 灵活定制 |

## 连线样式

支持 4 种连线渲染模式：

- **贝塞尔曲线** — 平滑优雅的曲线连接
- **直线** — 简洁直线连接
- **折线（阶梯）** — 正交折线连接
- **平滑折线** — 带圆角的折线

## 四向连接

每个节点提供上、下、左、右四个连接点，支持任意方向的连线。

## 编辑操作

| 操作 | 说明 |
|------|------|
| 拖拽移动 | 拖拽节点自由移动位置 |
| 连线 | 从连接点拖拽到目标节点 |
| 多选 | 框选或 Ctrl+Click 多选 |
| 复制粘贴 | Ctrl+C / Ctrl+V |
| 撤销重做 | Ctrl+Z / Ctrl+Y |
| 删除 | Delete 键删除选中元素 |
| 缩放 | 滚轮缩放画布 |
| 平移 | 拖拽空白区域平移画布 |

## 自动布局

点击工具栏「自动布局」按钮，FlowVision 使用 dagre 算法自动排列节点：

- 从上到下分层排列
- 自动避免节点重叠
- 保持连线清晰
