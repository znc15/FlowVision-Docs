# 斜杠命令

在聊天输入框中输入 `/` 即可调出命令面板，快速执行各种操作。

## 可用命令

| 命令 | 说明 |
|------|------|
| `/help` | 显示帮助信息 |
| `/clear` | 清空当前对话 |
| `/new` | 新建对话会话 |
| `/export` | 导出当前对话 |
| `/think` | 切换思考模式 |
| `/template` | 调用流程图模板 |

## 使用方式

1. 在聊天输入框输入 `/`
2. 弹出命令列表，可用方向键选择
3. 按回车执行选中命令
4. 部分命令支持额外参数

## 模板命令

`/template` 命令可快速调用预设的流程图模板：

- 用户登录流程
- 支付流程
- CI/CD 流水线
- 数据处理管线
- 更多模板持续添加中
