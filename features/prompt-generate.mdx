# AI Prompt 生成

描述你的场景，AI 帮你生成结构化、高质量的流程图 Prompt，提升生成效果。

## 使用场景

当你不确定如何描述需求时，Prompt 生成功能可以帮你：

- 将模糊描述转化为结构化提示词
- 补充遗漏的流程步骤
- 优化提示词以获得更好的生成效果

## 上下文支持

Prompt 生成支持两种上下文：

### 画布上下文

基于当前画布已有的流程图内容，生成扩展或优化的提示词。

### 项目上下文

基于已导入的项目代码结构，生成针对性的架构分析提示词。

## 工作流程

1. 点击「Prompt 生成」功能入口
2. 选择上下文类型（画布 / 项目）
3. 输入简要场景描述
4. AI 生成优化后的 Prompt
5. 点击「使用此 Prompt」一键调用
