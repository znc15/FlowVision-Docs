## Endpoint

```http
POST /plant/webhook
```

## 说明

这个页面保留为 webhook 文档结构示例，用来展示在纯静态站中如何书写事件回调说明。

## 请求头

| 头部 | 说明 |
| --- | --- |
| `Content-Type` | `application/json` |
| `X-Signature` | 事件签名 |

## 示例负载

```json
{
	"event": "plant.created",
	"timestamp": "2026-04-02T12:00:00Z",
	"data": {
		"id": "plant_003",
		"name": "Ficus elastica"
	}
}
```
