## Endpoint

```http
DELETE /plants/{id}
```

## 路径参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `id` | string | 是 | 要删除的植物 ID |

## 说明

删除指定植物记录。真实项目接入时，可以沿用当前页面结构，直接替换为 FlowVision 业务接口内容。

## 成功响应

```json
{
	"success": true,
	"deletedId": "plant_002"
}
```
