## Endpoint

```http
POST /plants
```

## 说明

创建一个新的植物条目。该页面用于替代原先依赖 Mintlify 自动渲染的 API 展示方式。

## 请求体

```json
{
	"name": "string",
	"price": 19.9,
	"category": "indoor"
}
```

## 成功响应

```json
{
	"id": "plant_002",
	"name": "string",
	"price": 19.9,
	"category": "indoor"
}
```

## 错误响应

```json
{
	"error": "unexpected error"
}
```
