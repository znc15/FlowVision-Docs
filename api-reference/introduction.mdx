## 当前结构

本目录包含两部分内容：

1. `api/` 目录中的接口说明页面
2. `api-reference/endpoint/` 目录中的示例端点页

这些页面现在都是普通 MDX 文档，不依赖平台专有的 OpenAPI 自动渲染能力，因此可以直接静态构建并部署到 GitHub Pages。
