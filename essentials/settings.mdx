当前项目不再使用 Mintlify 的 `docs.json`，而是改为 Docusaurus 官方配置文件 `docusaurus.config.ts`。

## 核心配置项

| 配置项 | 位置 | 说明 |
| --- | --- | --- |
| `title` | `docusaurus.config.ts` | 站点标题 |
| `tagline` | `docusaurus.config.ts` | 站点副标题 |
| `url` | `docusaurus.config.ts` | 生产站点域名 |
| `baseUrl` | `docusaurus.config.ts` | GitHub Pages 子路径 |
| `navbar.items` | `themeConfig.navbar` | 顶部导航 |
| `footer.links` | `themeConfig.footer` | 底部链接 |
| `prism` | `themeConfig.prism` | 代码高亮主题 |
| `staticDirectories` | 根配置 | 静态资源目录 |

## GitHub Pages 配置

```ts title="docusaurus.config.ts"
const repository = process.env.GITHUB_REPOSITORY || 'znc15/FlowVision-Docs';
const [organizationName, projectName] = repository.split('/');
const isUserSite = projectName.endsWith('.github.io');
const baseUrl = process.env.DOCS_BASE_URL || (isUserSite ? '/' : `/${projectName}/`);
```

这段配置会根据仓库名自动推导 GitHub Pages 的项目路径，避免手工维护不同环境的发布地址。

## 导航与页脚

```ts title="docusaurus.config.ts"
navbar: {
  title: 'FlowVision',
  items: [
    {to: '/', label: '文档', position: 'left'},
    {to: '/features/ai-generate', label: '功能指南', position: 'left'},
    {href: 'https://github.com/znc15/FlowVision', label: 'GitHub', position: 'right'},
  ],
}
```

```ts title="docusaurus.config.ts"
footer: {
  links: [
    {
      title: '文档',
      items: [{label: '快速开始', to: '/quickstart'}],
    },
  ],
}
```

## 主题与颜色

主要品牌色、卡片样式和页面背景位于 `src/css/custom.css`。如果你要调整整体视觉，优先修改：

- `:root` 下的 `--ifm-color-primary`
- `body` 背景渐变
- `.navbar-download-link`
- `.fv-link-card` 系列样式

## 实际维护建议

:::tip
如果你只是改页面内容，通常不需要碰 `docusaurus.config.ts`。只有站点级别的标题、导航、域名、主题或静态目录变化时，才需要修改这里。
:::

## 与旧结构的区别

:::warning
当前项目已经不再依赖 `docs.json`、`ResponseField`、`CodeGroup` 等 Mintlify 风格组件。后续新增文档应直接使用 Markdown、Tabs、details 和 Docusaurus 官方配置字段。
:::
