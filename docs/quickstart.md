# 快速开始

本页给你一条最短路径，用于在首次接触 FlowVision 时完成基础配置并生成第一张图。

## 1. 准备环境

- 获取 FlowVision 桌面版或开发环境
- 准备至少一个可用的 AI 提供方密钥
- 选择一个待分析的本地项目目录

## 2. 连接模型

在应用设置中配置模型提供方，例如 OpenAI、Anthropic、DeepSeek 或 Ollama。建议先使用响应速度稳定的模型完成首次体验。

## 3. 导入项目

打开项目分析入口，选择本地仓库。FlowVision 会扫描目录结构、关键模块和主要依赖关系，用于生成图谱初稿。

## 4. 生成流程图

输入一个明确目标，例如：

```text
生成后端请求链路图，突出网关、鉴权、服务层和持久化层。
```

## 5. 编辑与导出

在画布中补充说明、调整节点关系，然后选择导出 PNG、结构化数据或共享给其他协作端。
