# AI 生成

AI 生成功能负责把自然语言需求、项目上下文或已有图数据转换为可编辑的流程图结构。

## 输入来源

- 自然语言提示词
- 本地项目分析结果
- 现有图数据作为继续编辑的上下文

## 推荐提示词结构

1. 指定目标图类型
2. 指定关注边界
3. 指定输出重点

```text
生成订单支付链路图，突出支付网关、风控、订单服务和回调处理。
```

## 输出结果

- 节点与边的基础关系
- 模块说明和关键路径
- 可继续编辑的结构化图数据

## 使用建议

先让模型产出骨架，再在画布中人工修正关键节点，效率通常高于一次性追求完美结果。
