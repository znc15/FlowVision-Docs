# 配置指南

FlowVision 的配置分为四类：模型接入、分析策略、同步能力和导出行为。

## 模型配置

| 项目 | 说明 |
| --- | --- |
| Provider | OpenAI、Anthropic、DeepSeek、Ollama 等 |
| Model | 实际调用的模型名称 |
| API Key | 提供方认证凭据 |
| Base URL | 私有网关或兼容服务地址 |

## 分析配置

- 目录白名单与黑名单
- 忽略生成目录和缓存目录
- 分析深度与文件大小上限
- 是否提取依赖、接口和调用链

## 同步配置

- WebSocket 服务地址
- MCP 服务开关
- 自动保存间隔
- 冲突处理策略

## 导出配置

- 图片尺寸与背景色
- 节点标签显示策略
- JSON 数据版本
- 分享链接有效期

## 建议

首次部署时优先保证模型、分析路径和自动保存三项配置正确，其余项目可以在稳定使用后再按团队流程细化。
