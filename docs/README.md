## FlowVision 是什么？

FlowVision 是一个面向流程表达和系统理解的工作台。你可以把代码仓、需求或自然语言整理成流程图，然后继续在画布中编辑、协作，并通过 MCP 将能力开放给外部客户端或智能体。

## 推荐阅读顺序

1. [快速开始](./quickstart.md)：先完成安装并跑通基础流程。
2. [功能指南](./features/ai-generate.md)：了解 AI 出图、项目分析和画布编辑能力。
3. [MCP 集成](./mcp/overview.md)：查看客户端接入方式和工具能力边界。
4. [架构与 API](./architecture/overview.md)：在二次接入前确认系统边界和接口能力。
