# FlowVision

FlowVision 是一款面向研发团队的智能流程图平台，重点解决三类问题：

1. 把复杂系统快速转成结构化可读的图谱。
2. 让 AI、开发者和协作工具围绕同一份图状态工作。
3. 在生成、编辑、导出和同步之间保持一致的数据模型。

## 核心能力

| 能力 | 说明 |
| --- | --- |
| AI 自动生成 | 从自然语言提示或代码仓库生成流程图初稿 |
| 项目分析 | 自动识别模块、依赖、边界与关键流程 |
| 可视化编辑 | 在画布中拖拽、分组、自动布局和微调节点 |
| 实时同步 | 通过 WebSocket 与 MCP 共享图状态 |
| 导出与导入 | 支持图数据迁移、备份和团队交付 |

## 适用场景

- 新成员快速理解项目结构
- 研发方案评审和架构对齐
- AI 辅助生成业务流程图与技术流程图
- IDE、桌面端与自动化代理协同
