Set up Windsurf's Cascade AI assistant to help you write and maintain your Docusaurus documentation.

## Prerequisites

- Windsurf editor installed

## Setup

1. Open the repository root in Windsurf.
2. Review `AGENTS.md` and current docs structure before editing.
3. Use Cascade to draft or revise MDX pages.
4. Validate the output in the terminal:

```bash
pnpm build
```

5. Run a local preview when you need to inspect layout changes:

```bash
pnpm dev
```
