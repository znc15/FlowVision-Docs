Set up Claude Code to help you write and maintain your Docusaurus documentation.

## Prerequisites

- Active Claude subscription (Pro, Max, or API access)

## Setup

1. Install Claude Code

```bash
npm install -g @anthropic-ai/claude-code
```

2. Open this repository root in your terminal

The project already contains Docusaurus configuration, navigation settings, and documentation guidelines.

3. Review project instructions

Check `AGENTS.md` and the docs structure before editing content.

4. Start Claude Code

```bash
claude
```

5. Validate changes locally

```bash
pnpm build
```
