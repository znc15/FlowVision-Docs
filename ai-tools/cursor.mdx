Set up Cursor to help you write and maintain your Docusaurus documentation.

## Prerequisites

- Cursor editor installed

## Setup

1. Open the repository root in Cursor.
2. Read `AGENTS.md` to understand project terminology and writing constraints.
3. Edit MDX pages directly inside the docs tree.
4. Use the integrated terminal to validate the site:

```bash
pnpm build
```

5. Preview locally when needed:

```bash
pnpm dev
```
