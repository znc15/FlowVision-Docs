# 参与文档贡献

感谢你为 FlowVision 文档做出贡献。本仓库现在使用 VuePress Theme Hope 构建和发布文档站。

## 贡献方式

### 方式一：直接在 GitHub 上编辑

1. 打开你要修改的页面。
2. 点击页面对应文件的编辑按钮。
3. 提交变更并创建 Pull Request。

### 方式二：本地开发

1. Fork 并克隆仓库。
2. 安装依赖：`pnpm install`
3. 新建分支。
4. 修改 `docs/` 下的 Markdown 页面或 `.vuepress` 配置。
5. 启动本地预览：`pnpm docs:dev`
6. 在 `http://localhost:3000` 检查页面效果。
7. 提交变更并发起 Pull Request。

如需更多本地开发信息，请查看 [docs/development.md](docs/development.md)。

## 写作规范

- 使用主动语态。
- 直接面向读者写作。
- 句子尽量简洁，一句表达一个重点。
- 先写目标，再写操作步骤。
- 术语保持统一，优先使用仓库既有叫法。
- 关键步骤尽量提供示例。
